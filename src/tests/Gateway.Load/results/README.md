# Test results directory

This directory will contain the results of load test executions.

## Structure

Results are organized by:

- Date/time of execution
- Test type (smoke, load, stress, etc.)
- Environment (dev, staging, prod)

## File formats

- `*.json` - Raw k6 results in JSON format
- `*.html` - HTML reports with visualizations
- `*.csv` - CSV exports for further analysis
- `*.log` - Execution logs
